# vue-webpack

vue2.0 + vue-router2.0 + vue-resource1.0 + vue-element-ui2.0 + webpack

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
